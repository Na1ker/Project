# TradeTracker — статистика фьючерсных сделок BingX

Личный локальный трекер: подключается к BingX по API-ключу, каждую минуту
синхронизирует сделки, позиции и капитал, хранит всё в локальной SQLite и
показывает статистику за 7/30/90/180 дней, историю каждой сделки на графике
и заметки «почему зашёл».

Спека: [specs/bingx-trade-tracker.md](specs/bingx-trade-tracker.md)

## Запуск

Нужен Node.js 22+ (используется встроенный `node:sqlite`).

```bash
npm install
npm run dev
```

Открой http://localhost:3000 — при первом запуске появится онбординг с формой
API-ключей BingX (Профиль → API-менеджмент; достаточно прав «чтение»).

## Данные

- База: `data/tracker.db` (SQLite) — сделки, капитал, свечи, заметки. В git не попадает.
- API-ключи хранятся только в этой базе на твоём компьютере.
